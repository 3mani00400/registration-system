# Registration and Login System

## Description
This program allows users to register and login using a username, phone number, email address and password.

## Features
- Username validation (must contain "_" and ≤ 5 characters)
- Password validation (8+ chars, capital, number, special character)
- Phone number validation (10 digits)
- Login verification using HashMap

## How to Run
javac Registration.java
